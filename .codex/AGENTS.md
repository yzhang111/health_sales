# Project-Local ECC For Codex

This repository uses project-local ECC skills from `.agents/skills/`.

## Skills Discovery

- Skills are vendored into `.agents/skills/`.
- Each skill is self-contained under `.agents/skills/<skill-name>/`.
- Open the matching `SKILL.md` when a task clearly maps to that skill.

## Scope

- Treat these skills as repository-local.
- Do not assume they exist globally in `~/.codex/skills`.
- Prefer the project-local copy when both local and global versions exist.
